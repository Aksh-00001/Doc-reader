# Local Doc Reader Frontend

Mobile-first React PWA for local document upload, text preview, OCR output, and CSV/Excel table preview.

## Setup

```bash
cd frontend
npm install
cp .env.example .env
```

## Run

```bash
npm run dev
```

Open `http://localhost:5173`.

For Android testing, open the Vite network URL on your phone and use Chrome's **Add to Home screen** or **Install app** action.
